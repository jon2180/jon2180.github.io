    刷机那点儿事儿

    行也成文 11/30/2018 smartphone

    # 前奏

    • Q：怎么选择刷机包

    • A: 主要看需求，一般情况下需要考虑稳定性，易用性，主要优势和劣势，及特性（比如Gapps,xposed）.

    • Q: 刷机需要注意什么么？

    • A: 备份数据！备份数据！备份数据！

    # Step 1 备份数据及资源准备

    1. 主要备份项：

      • 应用列表
      • 联系人
      • 短信
      • 相册
      • 音乐
      • 通话记录
      • 便签
      • 各大平台的账号密码
      • 浏览器书签
      • 日历
      • 闹钟
      • WLAN
      • 重要的聊天记录

    2. 主要需要准备的资源

      • ROM包（压缩包或镜像）
      • Recovery镜像（通常TWRP）
      • 一些必备的应用安装包
      • 电脑
      • Bootloader解锁工具
      • adb工具

    3. 主要的第三方包

      • GSI - Generic System Image
      • 其他ROM

    # Step 2 解锁Bootloader

    不同手机的解锁方式不一样。这里列举小米手机的解锁工具

    1. 工具

      • MiUnlock

    2. 解锁步骤

      • 手机连接电脑
      • 打开MiUnlock，安装小米手机驱动
      • 登录账号，确认解锁

    # Step 3 刷入Recovery（线刷忽略此步骤）

    1. 使用手机软件刷入

      1. 获取ROOT权限
      2. 安装Flashfiy
      3. 刷入Recovery镜像
      4. 重启

    2. 使用电脑软件刷入

      • 工具不同，方法不同

    3. 使用ADB(Android Debug Bridge)工具

      1. 手机连接电脑，打开usb调试
      2. 打开cmd，进入adb文件的路径
      3. 检查是否连接成功
        adb devices
        1
      4. 刷入镜像
        adb flash recovery <file path and file name>
        1

    # Step 4 刷入ROM

    1. 通过REC模式刷入
      1. 将安装包放在根目录
      2. 选中刷入，一般双清，特殊情况特殊考虑
    2. 通过ADB输入
      1. 手机连接电脑，打开usb调试
      2. 打开cmd，进入adb文件的路径
      3. 检查是否连接成功
        adb devices
        1
      4. 刷入镜像
        adb flash system <file path and file name>
        1

    # Step 5 重启
